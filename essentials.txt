 ReactJS – Introduction and Features
 1. Understanding ReactJS

ReactJS is a JavaScript library used to build user interfaces (UI), especially for Single Page Applications (SPA).

React builds UI using components.

Example:

```javascript
function Hello() {
  return <h1>Hello World</h1>;
}
```

Here **Hello** is a component.
 Why React?

* Fast rendering
* Reusable components
* Easy to maintain
* Virtual DOM improves performance

# 3. Using NPM

Install package

```bash

```

Create React project

```bash

```

Start project

```bash
npm start
```

# 8. Understanding React Syntax

React uses **JSX (JavaScript XML)** syntax.

JSX looks like HTML but inside JavaScript.

Example:

```javascript
const element = <h1>Hello React</h1>;
```

Without JSX:

```javascript
const element = React.createElement('h1', null, 'Hello React');
```

JSX makes code **clean and readable**.

---

# 9. Understanding React-dom.js

**React DOM** connects **React components with the browser DOM**.

It renders React elements into the webpage.

Example:

```javascript
import React from "react";
import ReactDOM from "react-dom";

ReactDOM.render(
  <h1>Hello React</h1>,
  document.getElementById("root")
);
```

Here ReactDOM displays the component in the browser.

---

# 10. Understanding Babel

**Babel** is a **JavaScript compiler**.

It converts modern JavaScript and JSX into browser-compatible JavaScript.

Example

JSX code:

```javascript
const element = <h1>Hello</h1>;
```

Babel converts it into:

React.createElement("h1", null, "Hello");

This allows React to run in all browsers.

# ReactJS Features

Main Features of React:
1. Virtual DOM
Improves performance by updating only changed parts.
2. Component Based
UI is divided into reusable components.
3. One-way Data Binding
Data flows from parent → child components.
4. JSX
Makes UI code simple and readable.
5. High Performance
Fast rendering and efficient updates.
